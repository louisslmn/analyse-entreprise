# Notes

## Pourquoi un score par rang ?
Une normalisation min/max peut être très sensible aux valeurs extrêmes.
Le score par rang (percentile) est souvent plus robuste :
- on compare “relativement” les entreprises
- on obtient un score 0..100 interprétable

## Dimension extra‑financière
On intègre 3 variables :
- intensité carbone (min = mieux)
- part de CA vert (max = mieux)
- score ESG (max = mieux)

Dans la vraie vie :
- les définitions exactes varient selon les providers
- la donnée peut être manquante / bruitée

Ici, le but est de montrer la méthode (et l’intérêt du sujet).
